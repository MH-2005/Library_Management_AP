# 📚 Library Management System

یک سیستم مدیریت کتابخانه مبتنی بر خط فرمان (CLI) که با **جاوا** و با تمرکز بر اصول **برنامه‌نویسی شیءگرا (OOP)** پیاده‌سازی شده است.

> ⚠️ **نکته:** این پروژه یک **تمرین درس «برنامه‌نویسی پیشرفته»** است که قبلاً انجام داده‌ام. هدف اصلی آن تمرین و به‌کارگیری مفاهیم شیءگرایی (وراثت، چندریختی، کپسوله‌سازی، interface و کلاس‌های انتزاعی) بوده و نه یک محصول آمادهٔ production.

## 🎯 دربارهٔ پروژه

این سیستم قابلیت‌های کامل یک کتابخانه را شبیه‌سازی می‌کند:

- 👤 مدیریت کاربران (دانشجو، استاد، کارمند، مدیر کتابخانه)
- 📖 مدیریت منابع (کتاب، پایان‌نامه، کتاب فروشی، کتاب گنجینه)
- 🔄 امانت‌دادن و پس‌گرفتن منابع
- 🛒 فروش کتاب‌های فروشی
- 🔎 جستجوی منابع و کاربران
- 📊 گزارش‌گیری‌های گوناگون (گزارش کتابخانه، محبوب‌ترین‌ها، فروش، جریمه‌ها و ...)

ورودی و خروجی برنامه از طریق **کنسول** و با یک فرمت دستور متنی یکپارچه انجام می‌شود.

## 🧠 مفاهیم به‌کاررفته

این پروژه برای تمرین مفاهیم درس **برنامه‌نویسی پیشرفته** طراحی شده و از موارد زیر استفاده می‌کند:

| مفهوم | محل استفاده |
|------|-------------|
| **کلاس انتزاعی (Abstract Class)** | `User`، `Admin` |
| **وراثت (Inheritance)** | `Student`، `Staff`، `Master`، `Manager` ← `User` |
| **چندریختی (Polymorphism)** | متدهای `getBorrowLimit()`، `getType()`، `matchesSearch()` |
| **Interface** | `Borrowable` |
| **کپسوله‌سازی (Encapsulation)** | فیلدهای `private` + getter/setter |
| **Composition / HashMap** | نگه‌داری منابع، کاربران، دسته‌بندی‌ها |
| **API تاریخ‌و‌زمان جاوا** | `java.time` برای محاسبهٔ مدت امانت و جریمه |

## 🗂️ ساختار پروژه

```
Library_Management_AP/
├── src/
│   └── Main.java          # تمام کلاس‌های سیستم (یک فایل واحد)
├── .gitignore
├── CONTRIBUTING.md
├── LICENSE
└── README.md
```

> تمام کلاس‌ها در یک فایل `Main.java` قرار دارند تا مطابق با ساختار خواسته‌شدهٔ تمرین باشد.

### کلاس‌های اصلی

- **`Main`** — نقطهٔ ورود برنامه و حلقهٔ خواندن دستورات
- **`CommandHandeling`** — تجزیه و مسیریابی دستورها
- **`Admin`** / **`Manager`** — احراز هویت
- **`Library`** — مدیریت منابع، امانت، گزارش‌گیری
- **`User`** (abstract) ← `Student`, `Staff`, `Master`, `Manager`
- **`Resource`** ← `Book`, `Thesis`, `SellingBook`, `TreasureBook`
- **`Category`** — دسته‌بندی درختی منابع
- **`Borrowable`** (interface) — برای منابع قابل‌امانت

## 🚀 نحوهٔ اجرا

```bash
# ۱. کامپایل
javac -d out src/Main.java

# ۲. اجرا
java -cp out Main
```

سپس برنامه منتظر دریافت دستور می‌ماند. برای پایان، دستور `finish` را وارد کنید.

---

## ⌨️ فرمت دستورات

تمام دستورها با الگوی زیر وارد می‌شوند:

```
action#param1#param2#param3...
```

- جداکنندهٔ بخش‌ها: کاراکتر `#`
- جداکنندهٔ تاریخ‌و‌زمان: کاراکتر `|` (مثلاً `2023-01-15|10:30`)
- پایان برنامه: دستور `finish`

## 📜 لیست دستورات

### مدیریت سیستم (توسط ادمین)
| دستور | پارامترها (بعد از ادمین) |
|-------|--------------------------|
| `add-library` | `username#pass#id#name#year#deskCount#address` |
| `add-category` | `username#pass#id#name#parentCategoryId` |
| `add-student` | `username#pass#id#pass#firstName#lastName#nationalCode#birthYear#address` |
| `add-staff` | `username#pass#id#pass#firstName#lastName#nationalCode#birthYear#address#type` (`type`: `staff` یا `professor`) |
| `add-manager` | `username#pass#id#pass#firstName#lastName#nationalCode#birthYear#address#libraryId` |
| `remove-user` | `username#pass#userId` |
| `report-penalties-sum` | `username#pass` |

> 📌 حساب پیش‌فرض ادمین: نام کاربری `admin` — رمز عبور `AdminPass`

### مدیریت منابع (توسط مدیر کتابخانه)
| دستور | توضیح |
|-------|-------|
| `add-book` | افزودن کتاب قابل امانت |
| `add-thesis` | افزودن پایان‌نامه |
| `add-ganjineh-book` | افزودن کتاب گنجینه (فقط مطالعه در محل) |
| `add-selling-book` | افزودن کتاب فروشی |
| `remove-resource` | حذف منابع |

### عملیات کاربران
| دستور | توضیح |
|-------|-------|
| `borrow` | امانت‌گرفتن منبع |
| `return` | پس‌دادن منبع |
| `buy` | خرید کتاب فروشی |
| `read` | مطالعهٔ کتاب گنجینه (فقط استاد) |
| `add-comment` | افزودن نظر به منبع |
| `search` | جستجوی منابع |
| `search-user` | جستجوی کاربران |

### گزارش‌گیری
| دستور | توضیح |
|-------|-------|
| `library-report` | آمار منابع یک کتابخانه |
| `category-report` | آمار منابع یک دسته |
| `report-passed-deadline` | امانت‌های عبورکرده از مهلت |
| `report-sell` | گزارش فروش کتاب‌ها |
| `report-most-popular` | محبوب‌ترین منابع |
